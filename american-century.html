1 Discover

Understanding the legacy landscape

American Century Investments needed to modernize their digital presence across web and mobile platforms. The existing system relied on a legacy CMS with inconsistent components, making it difficult for content teams and developers to work efficiently. I began by auditing the existing client portal, landing pages, and internal ESG dashboard to map pain points across user journeys.

Key Finding

The lack of a unified design system meant each platform (Android, iOS, desktop web) had divergent UI patterns — creating confusion for users who used multiple devices and slowing down the engineering team significantly.

3 Design

Building a cross-platform design system in Figma

I led the creation of a comprehensive design system that unified the experience across Android, iOS, and desktop web. Every component was built in Figma with auto-layout, variants, and detailed documentation for developer handoff. Key features I designed included the Transaction Confirmation flow, client dashboard, and investment tools — each requiring careful attention to accessibility and financial data visualization.

Design System Impact

The unified component library eliminated duplicate work across platforms, allowing engineers to reference a single source of truth — reducing design-to-dev handoff time and ensuring pixel-perfect consistency across 3 platforms.

4 Iterate

Cross-functional reviews with PM & Engineering

Each feature went through multiple rounds of review with product managers and engineering leads. For the CMS migration specifically, I worked hand-in-hand with front-end developers to ensure every Contentful component matched the design specs for sizing, typography, spacing, and responsive behavior. Regular critiques helped us catch edge cases early — like how transaction confirmation modals behaved on smaller screens.

5 Handoff

Developer-ready specs & Figma documentation

Every component shipped with detailed redline specs, interaction notes, and responsive breakpoints documented directly in Figma. I created handoff documentation that mapped each design system component to its Contentful content model, making it easy for developers to implement components that were both visually consistent and content-flexible.
